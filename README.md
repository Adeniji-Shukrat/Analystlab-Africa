# AnalystLab Africa - Week 1-2 Task: Data Cleaning & EDA

## Overview
This repository contains the data cleaning and exploratory analysis for the 
E-commerce Transactions dataset and Netflix Movies & TV Shows dataset, completed 
as part of the AnalystLab Africa Data Analytics Internship (Batch B).

## Contents
- `Cleaned_Online_Retail.zip` — Cleaned E-commerce dataset (CSV, zipped due to file size)
- `OnlineRetail_AnalystLab.ipynb` — Full notebook: cleaning, EDA, visualizations, insights (E-commerce)
- `netflix_analysis.ipynb` — Full notebook: cleaning, EDA, visualizations, insights (Netflix)
- `cleaned_netflix.csv` — Cleaned Netflix dataset

## Tools Used
Python (Pandas, Matplotlib, Seaborn) via Google Colab

## Note
The cleaned e-commerce CSV is provided as a zip file due to GitHub's 25MB upload limit — unzip to access the file.
